# CV Feedback App 

App web que analiza CVs con Inteligencia Artificial y genera feedback detallado para mejorar tu perfil profesional.

## ¿Qué hace?
- Subís tu CV en formato PDF
- La IA analiza el contenido
- Recibís feedback detallado sobre fortalezas, debilidades y cómo mejorar

## Tecnologías usadas
- **Frontend**: React.js
- **Backend**: Node.js + Express
- **IA**: Groq (LLaMA 3.3 70B)
- **Procesamiento de PDF**: pdf2json

## Cómo correrlo localmente

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start
